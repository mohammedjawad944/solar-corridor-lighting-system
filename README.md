# Sustainable Corridor Lighting Solution Using Solar Panels

This project implements an automated corridor lighting system powered by solar energy. The system uses motion detection to turn lights ON only when a person is present, reducing electricity consumption.

## Components
ESP32
PIR Motion Sensor
Solar Panel
Rechargeable Battery
LED Lights

## Features
Motion-based automatic lighting
Energy efficient corridor lighting
Solar charging system with battery backup
Reduces electricity consumption

## Technologies Used
Embedded Systems
IoT
Microcontrollers

## Applications
College corridors
Office buildings
Smart campuses
